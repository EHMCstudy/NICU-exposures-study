# Metadata Directory

This directory contains de-identified metadata and summary tables prepared from `Tables_supplemental-revision.xlsx`, focusing on `Table S1` and `Table S2`.

## Files

- `table_s1_infant_summary.csv`: cohort-level summary statistics derived from Supplementary Table S1.1
- `table_s1_infant_metadata.csv`: de-identified infant-level metadata derived from Supplementary Table S1.2
- `table_s2_sample_counts.csv`: assay-level sample counts derived from Supplementary Table S2.1
- `table_s2_sample_metadata.csv`: de-identified sample-level metadata derived from Supplementary Table S2.2
- `data_dictionary.md`: concise description of the released metadata files

Do not upload protected or identifiable source data here.
