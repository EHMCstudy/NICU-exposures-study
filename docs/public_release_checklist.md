# Public Release Checklist

## Before Submission

- Confirm that the public GitHub repository contains the final analysis script(s).
- Confirm that every repository URL used in the manuscript resolves publicly.
- Confirm that the public metabolomics repository link resolves with HTTP 200.
- Confirm that the sample-level metadata file is approved for public release.
- Add accession crosswalk columns to the public metadata table if available (`SRR`, `SRS`, `SAMN`).
- Create a GitHub release or tag for the manuscript-associated version.

## Nice-to-Have

- Add a software license.
- Add an environment lockfile or package session manifest.
- Add a short data dictionary for the public metadata table.
- Archive the GitHub release in Zenodo for DOI-based software citation.
